Most public tooling assumes qubits by default. This project focuses on building a usable software layer for qutrit circuits, gate composition, and experimentation with higher-dimensional state spaces.

The longer-term aim is to make qutrit circuit exploration feel as direct as standard qubit workflows while preserving the algebraic differences that matter.

Near-term goals:

- define core gate primitives and composition rules
- support statevector simulation for small systems
- build examples that expose where qutrit reasoning differs from qubit intuition
